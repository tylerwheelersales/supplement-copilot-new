Milestone 1 (MVP):

User can create an account and log in.

User can add supplements (name, brand, servings per bottle, servings per day, Amazon link).

User can add a supplement by taking or uploading a photo of the bottle. The photo is analyzed by AI (Claude vision) to automatically extract and pre-fill supplement details (brand name, type, servings per bottle, servings per day, etc.). User can review and edit before saving.

User can log when they take a supplement.

App tracks remaining servings and estimates a run-out date.

When remaining servings drop below a threshold, app shows a low-stock warning and a button to open the Amazon link.
